# Квіз - Платформа для створення тестів

Сайт квіз з шаблонами тесту різних видів, побудований на Next.js.

## Технології

- Next.js 15
- React 19
- TypeScript
- CSS Modules

## Встановлення

```bash
npm install
```

## Запуск проекту

```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Структура проекту

```
src/
  components/     # React компоненти
  pages/          # Сторінки Next.js
  styles/         # Глобальні стилі
  types/          # TypeScript типи
  lib/            # Утиліти та хуки
```

## Доступні сторінки

- `/` - Головна сторінка
- `/quiz` - Список квізів
- `/templates` - Шаблони квізів
- `/about` - Про нас
- `/contact` - Контакти
